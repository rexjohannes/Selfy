# Selfy
A discord selfbot.

##Download
1. Download the latest release
2. Open your command line in the folder of Selfy.jr
3. Type in `java -jar Selfy.jar <Your Discord Token>`

##Find your Discord Token

1. Press Ctrl + Shift + I
2. Open Application
3. Expans Local Storage and select discord.com
4. Search for `token`
5. Press `Ctrl + R`
6. Copy your token

##Notes

The default prefix is: **.**

For help type in **.help**
