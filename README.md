# Selfy
A discord selfbot.

## Download

1. Download the latest release
2. Open your command line in the folder of Selfy.jr
3. Type in `java -jar Selfy.jar <Your Discord Token>`

## Find your Discord Token

1. Open Discord in your Browser amd login
2. Press `Ctrl + Shift + I`
3. Reload the site
4. Open Application
5. Expans Local Storage and select discord.com
6. Search for `token`
7. Reload the website
8. Copy your token

## Notes

The default prefix is: **.**

For help type in. **.help**
